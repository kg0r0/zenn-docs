# 事前準備

まず、事前に[Google API Console](https://console.cloud.google.com/apis/dashboard)からOAuth2.0のクライアント登録を実施し、以下の3つの情報を取得しておく必要があります。
- Client ID
- Client Secret
- Authorized redirect URIs

# 設定

```bash
$ git clone -b develop https://github.com/kg0r0/envoy-oauth2-filter-google.git
$ cd envoy-oauth2-filter-google
```

```bash
$ docker-compose up -d
```

# 動作確認

``http://localhost:40080``にアクセスします。
